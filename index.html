Party Limit Begone

Installation Guide for Baldur's Gate 3 (Patch 8 Compatible)

What Does This Mod Do?

Party Limit Begone removes the 4-party-member restriction in Baldur's Gate 3, allowing you to adventure with:

  • Up to 16 companions in single-player
  • Up to 8 players in multiplayer co-op

Now your whole friend group can play together without anyone being left out!

Installation (Everyone Needs to Do This)

All players need to install the mod using BG3 Mod Manager. Vortex also works if you have Script Extender installed.

1 Download and Setup BG3 Mod Manager
  1. Download BG3 Mod Manager
  2. Extract and run it

[GIF: Setting up BG3 Mod Manager]

2 Install Script Extender
  1. In the mod manager, go to Tools
  2. Select "Download & Install the Script Extender"
  3. Click Yes when prompted

[GIF: Installing Script Extender]

3 Download and Import the Mod
  1. Download Party Limit Begone from Nexus Mods
  2. Drop the zip file directly into your mod manager window

[GIF: Dropping zip into mod manager]

4 Activate the Mod
  1. Go to File → Reload All
  2. Drag and drop Party Limit Begone from the right side to the left (Active Mods)
  3. Go to File → Save Order

[GIF: Activating mod and saving order]

Done!

Launch the game and enjoy up to 16 companions!

Tip: If you already have 4 party members, dismiss one and re-add them to reset the party limit to 16.

Want 5-8 players in multiplayer? The HOST needs to continue to the Multiplayer Patch section below.

Multiplayer Patch (Host Only - Required for 5+ Players)

If you want more than 4 players in multiplayer, the HOST needs to apply this additional patch after completing the installation above.

1 Download the Multiplayer Patch
  1. Go to Party Limit Begone on Nexus Mods
  2. Download the Multiplayer Patch file
  3. Unzip it
2 Run the Patcher

Drag and drop either your bg3.exe OR bg3_dx11.exe onto the PartyLimitBegonePatcher.bat file.

EXE location: \steamapps\common\Baldurs Gate 3\bin\

The patcher will create a backup for you automatically.

Which EXE to use?

  • bg3.exe - If you use Vulkan in the Larian launcher
  • bg3_dx11.exe - If you use DirectX 11 in the Larian launcher

[GIF: Dragging exe onto patcher]

Windows Defender Warning

Windows Defender might flag the patcher on some systems - this is safe to use. Nexus Mods scans every upload. Just allow it and continue.

Done!

You can now play with up to 8 players in multiplayer!

How to Launch the Game

Important: Steam Matchmaking Issues

The latest hotfix broke Steam matchmaking for some users. If you're having connection issues, use the workaround below.

Recommended: Launch bg3.exe Directly

Instead of launching through Steam, close Steam and launch the game directly from the executable:

C:\Program Files (x86)\Steam\steamapps\common\Baldurs Gate 3\bin\bg3.exe

This bypasses Steam matchmaking and can fix connection issues.

Tip: Create a desktop shortcut to bg3.exe for easy access!

Alternative Launch Methods

Method How Notes
Direct EXE Run bg3.exe from the bin folder Recommended if having issues
Steam with --skip-launcher Add --skip-launcher to Steam launch options Skips Larian launcher
Normal Steam Launch through Steam as usual Works for most users

Multiplayer Setup

For the Host

If you want to expand an existing multiplayer save to use larger parties:

  1. Create a new lobby
  2. Set your desired party size in the settings
  3. Start a new game
  4. Then load your existing save

You cannot change the party size limit from within an active game session.

For Players Joining

Simply join the host's game as normal. No installation required! If you see a warning about data mismatch, just click through it.

Console/Crossplay Support

Good News!

The Standalone version works with console players if the hosting player is on PC!

Limitations

  • Only 3 console players are supported
  • Console players must join first because only PC players can join after 4 slots are taken
  • Setup: Host (PC) + 3 console players first, then PC players 5+ join after

Player 5+ Connection Methods

Players 5 and above can only join via:

Method How
LAN Mode (Hamachi) Host and player 5+ must be in the same Hamachi network. Host enables "LAN connection" in Options → Gameplay.
Direct IP Host opens port 23253, then players type the host's IP in the direct code box to join.

Continuing an Existing Crossplay Save

To continue an existing crossplay save with more than 4 players:

  1. Create a new fresh lobby
  2. Have everyone join
  3. Start a new game
  4. Once in-game, load your old main save

Note: This creates new save files, so the host should clear them out periodically.

Linux and Mac Support

PLB Works on Linux and Mac!

The standalone version is supported on both Linux and Mac.

MP Patch Installation is Tricky

The multiplayer patch requires extra steps on Linux/Mac.

Easiest Solution

Ask a friend on Windows to send you their patched exe file after they run the patcher.

Manual Method (with Wine)

  1. Open PartyLimitBegonePatcher.bat with any text editor
  2. Find this line: Set "GetFileName=%~n1.exe"
  3. Change it to: Set "GetFileName=bg3.exe"
  4. Place your bg3.exe file in the same folder as the patcher
  5. Run PartyLimitBegonePatcher.bat with Wine

Troubleshooting

Common Fixes (from Mod Author)

  1. Delete your .exe files and let Steam verify the game files (Right-click game → Properties → Local Files → Verify). Then reapply the MP patch if needed.
  2. Delete your settings folder (keep PlayerProfiles!):
    C:\Users\%username%\AppData\Local\Larian Studios\Baldur's Gate 3

    Delete everything in this folder except the PlayerProfiles folder.

Mod isn't working?

  • Try dismissing one of your current party members, then re-add them. This resets the party limit.
  • Make sure you exported the load order in BG3 Mod Manager
  • Verify Script Extender is installed (Method 1)

Can't connect to multiplayer?

  • Close Steam and launch bg3.exe directly (see How to Launch)
  • This bypasses Steam matchmaking which is broken for some users

Game won't launch?

  • If you used manual installation, restore your backup bg3.exe
  • Verify game files through Steam (Right-click game → Properties → Local Files → Verify)

Data mismatch warning?

  • This is completely normal - just click through it!

Known Issues (READ THIS!)

Patch 8 Specific Bugs

  • Honor Mode Issue: Fresh lobbies with more than 4 players cause problems. Workaround: Launch with 4 or fewer players, then invite the rest once you're fully in-game.
  • Crossplay Lobbies: No longer display the direct connection ID. Playing with more than 4 players still works using LAN mode with Hamachi.
  • Steam Matchmaking: Latest hotfix broke it for some users. Workaround: Close Steam and launch bg3.exe directly.

General Known Bugs

  • Grymforge Map Transition: Reduce your party size to 4 members before sailing with the boat. If characters are stuck, dismiss the ones with you and sail back and forth to unstick the rest. Note: After using the boat, a timed quest is triggered and going to camp will progress it.
  • Quest/Important Items Bug: Some quest or important items aren't registered correctly if carried by a 5th or higher party member. Make sure your first 4 party members carry any important items!
  • Camp Sleeping Bug: Use one of the awake companions and talk to Lae'zel, use a camp chest, or reduce your party size to 4 before resting.
  • Hardcoded Events: Some in-game events are hardcoded for 4 party members and will glitch out. This is usually solved by dismissing the additional members/players or by teleporting back and forth.

Uninstalling the Mod

If using BG3 Mod Manager:

  1. Drag the mod from Active to Inactive mods
  2. Save and export load order

If installed manually:

  1. Restore your backup bg3.exe file
  2. Delete the Mods folder from your Data directory
