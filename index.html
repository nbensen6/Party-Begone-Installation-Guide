Party Limit Begone

Installation Guide for Baldur's Gate 3 (Patch 8 Compatible)

What Does This Mod Do?

Party Limit Begone removes the 4-party-member restriction in Baldur's Gate 3, allowing you to adventure with:

  • Up to 16 companions in single-player
  • Up to 8 players in multiplayer co-op

Now your whole friend group can play together without anyone being left out!

Do All Players Need the Mod?

Good News for Your Friends!

Only the HOST needs to install the mod. Other players can join the session without installing anything!

Player Role Mod Required? Notes
Host (creates the game) Yes Must have mod fully installed
Joining Players No Can join without any installation

Important Note

The game launcher may show a "data mismatch" warning when joining. This is completely normal and can be safely ignored - just click through it!

Method 1: BG3 Mod Manager (Recommended)

This is the easiest method for beginners. The mod manager handles everything for you!

1 Download and Install BG3 Mod Manager
  1. Go to the BG3 Mod Manager releases page
  2. Download the latest .zip file
  3. Extract it to a folder (NOT in Program Files)
  4. Run BG3ModManager.exe

[GIF: Downloading and extracting BG3 Mod Manager]

2 Install Script Extender

Party Limit Begone requires Script Extender to work.

  1. In BG3 Mod Manager, go to Tools menu
  2. Click "Download & Extract the Script Extender"
  3. Wait for it to complete

[GIF: Installing Script Extender from Tools menu]

3 Download Party Limit Begone
  1. Go to Party Limit Begone on Nexus Mods
  2. Click the "Files" tab
  3. Click "Manual Download" on the main file
  4. Save the .zip file somewhere you can find it

[GIF: Downloading from Nexus Mods]

4 Import the Mod
  1. In BG3 Mod Manager, go to File → Import Mod
  2. Select the .zip file you downloaded
  3. The mod will appear in the right panel (Inactive Mods)

[GIF: Importing mod into BG3 Mod Manager]

5 Activate the Mod
  1. Drag the mod from the right panel to the left panel (Active Mods)
  2. Click the Save icon (or press Ctrl+S) to save your load order
  3. Click "Export Load Order to Game" button

[GIF: Dragging mod to active and exporting]

6 Launch the Game
  1. Start Baldur's Gate 3
  2. If you see a "data mismatch" warning, click through it
  3. Load your save or start a new game
  4. Dismiss one party member, then re-add them to activate the new limit

[GIF: Launching game and dismissing/re-adding party member]

You're Done!

The mod is now installed. You can have up to 16 companions or 8 players in your party!

Method 2: Manual Installation

If you prefer not to use a mod manager, you can install manually.

1 Download the Mod
  1. Go to Party Limit Begone on Nexus Mods
  2. Download the main file
  3. Extract the .zip file
2 Move Files to Game Folder

Move the entire Mods folder from the extracted files into your BG3 Data directory:

C:\Program Files (x86)\Steam\steamapps\common\Baldurs Gate 3\Data

Note: Your Steam path may be different if you installed Steam elsewhere.

[GIF: Moving Mods folder to Data directory]

3 Run the Patcher
  1. Find PartyLimitBegonePatcher.bat in the extracted files
  2. Find your bg3.exe file (in the game's bin folder)
  3. Drag and drop bg3.exe onto the .bat file
  4. A backup will be created automatically
bg3.exe location: C:\Program Files (x86)\Steam\steamapps\common\Baldurs Gate 3\bin\bg3.exe

[GIF: Dragging bg3.exe onto the patcher]

4 Launch and Verify
  1. Start Baldur's Gate 3
  2. Dismiss the "data mismatch" warning if it appears
  3. Dismiss and re-add a party member to activate the new limit

How to Launch the Game

Important: Steam Matchmaking Issues

The latest hotfix broke Steam matchmaking for some users. If you're having connection issues, use the workaround below.

Recommended: Launch bg3.exe Directly

Instead of launching through Steam, close Steam and launch the game directly from the executable:

C:\Program Files (x86)\Steam\steamapps\common\Baldurs Gate 3\bin\bg3.exe

This bypasses Steam matchmaking and can fix connection issues.

Tip: Create a desktop shortcut to bg3.exe for easy access!

Alternative Launch Methods

Method How Notes
Direct EXE Run bg3.exe from the bin folder Recommended if having issues
Steam with --skip-launcher Add --skip-launcher to Steam launch options Skips Larian launcher
Normal Steam Launch through Steam as usual Works for most users

Multiplayer Setup

For the Host

If you want to expand an existing multiplayer save to use larger parties:

  1. Create a new lobby
  2. Set your desired party size in the settings
  3. Start a new game
  4. Then load your existing save

You cannot change the party size limit from within an active game session.

For Players Joining

Simply join the host's game as normal. No installation required! If you see a warning about data mismatch, just click through it.

Cross-Play Note

If playing with console friends: Have console players join first, then PC players join after. This helps avoid connection issues.

Troubleshooting

Common Fixes (from Mod Author)

  1. Delete your .exe files and let Steam verify the game files (Right-click game → Properties → Local Files → Verify). Then reapply the MP patch if needed.
  2. Delete your settings folder (keep PlayerProfiles!):
    C:\Users\%username%\AppData\Local\Larian Studios\Baldur's Gate 3

    Delete everything in this folder except the PlayerProfiles folder.

Mod isn't working?

  • Try dismissing one of your current party members, then re-add them. This resets the party limit.
  • Make sure you exported the load order in BG3 Mod Manager
  • Verify Script Extender is installed (Method 1)

Can't connect to multiplayer?

  • Close Steam and launch bg3.exe directly (see How to Launch)
  • This bypasses Steam matchmaking which is broken for some users

Game won't launch?

  • If you used manual installation, restore your backup bg3.exe
  • Verify game files through Steam (Right-click game → Properties → Local Files → Verify)

Data mismatch warning?

  • This is completely normal - just click through it!

Known Issues (Patch 8)

Patch 8 Specific Bugs (from Mod Author)

  • Honor Mode Issue: Fresh lobbies with more than 4 players cause problems. Workaround: Launch with 4 or fewer players, then invite the rest once you're fully in-game.
  • Crossplay Lobbies: No longer display the direct connection ID. Playing with more than 4 players still works using LAN mode with Hamachi.
  • Steam Matchmaking: Latest hotfix broke it for some users. Workaround: Close Steam and launch bg3.exe directly.

General Issues

  • Some events may glitch with 5+ party members. Solution: Temporarily dismiss extra members for that encounter.
  • Grymforge boat: Reduce party to 4 members before sailing.
  • Camp sleeping bugs: Use awake companions or reduce party to 4.
  • Some cutscenes are designed for 4 players and may look odd with more.

Uninstalling the Mod

If using BG3 Mod Manager:

  1. Drag the mod from Active to Inactive mods
  2. Save and export load order

If installed manually:

  1. Restore your backup bg3.exe file
  2. Delete the Mods folder from your Data directory
