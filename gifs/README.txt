GIF FILES FOR INSTALLATION GUIDE
================================

Add your GIF files to this folder with the following names:

For Method 1 (BG3 Mod Manager):
- step1-download-manager.gif    : Downloading and extracting BG3 Mod Manager
- step2-script-extender.gif     : Installing Script Extender from Tools menu
- step3-download-mod.gif        : Downloading mod from Nexus Mods
- step4-import-mod.gif          : Importing mod into BG3 Mod Manager
- step5-activate-mod.gif        : Dragging mod to active and exporting load order
- step6-launch-game.gif         : Launching game and dismissing/re-adding party member

For Method 2 (Manual Installation):
- manual-step2-move-files.gif   : Moving Mods folder to Data directory
- manual-step3-patcher.gif      : Dragging bg3.exe onto the patcher

RECOMMENDED GIF SETTINGS:
- Width: 800px max (for good display on all screens)
- Frame rate: 10-15 fps (keeps file size manageable)
- Duration: 5-15 seconds per GIF
- Use a tool like ScreenToGif, ShareX, or LICEcap to record

After adding your GIFs, update index.html to uncomment the <img> tags
and remove the placeholder text.
